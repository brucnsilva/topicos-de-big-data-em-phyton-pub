ARQUIVO: Estatísticas de segurança: série histórica mensal por município desde 2014

O objetivo deste trabalho foi analisar os crimes de roubo e furto de veículos no Estado do Rio de Janeiro entre 2014 e 2026 utilizando técnicas de Big Data e análise estatística. A partir de dados públicos do ISP-RJ, foram aplicadas análises descritivas, regressão linear, distribuição de frequência e identificação de outliers para compreender a distribuição dos crimes e sua relação com as ocorrências de tráfico de drogas.
